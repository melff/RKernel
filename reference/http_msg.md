# Handle a generic message request sent via getting from an url with the slug 'msg'.

Handle a generic message request sent via getting from an url with the
slug 'msg'.

## Usage

``` r
http_msg(path, query, ...)
```

## Arguments

- path:

  The url, excluding the hostname and port, but including the slug
  "data" and the name of an R object.

- query:

  The query string translated into a character vector.

- ...:

  Any other arguments, ignored.
