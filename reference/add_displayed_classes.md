# Add a Class to the 'Displayed' Ones

Add a class to those who are output using
[`display()`](https://melff.github.io/RKernel/reference/display.md) when
they are autoprinted, i.e. returned as the value of the last expression
in a Jupyter notebook cell.

## Usage

``` r
add_displayed_classes(x)
```

## Arguments

- x:

  A character string, the name of a class.
