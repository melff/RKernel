# Start interactive help system

A variant of [`help.start`](https://rdrr.io/r/utils/help.start.html)
that works when called from inside a Jupyter notebook.

## Usage

``` r
help.start(
  update = FALSE,
  gui = "irrelevant",
  browser = getOption("browser"),
  remote = NULL
)
```

## Arguments

- update:

  A logical value. This formal argument exists for compatibility reasons
  only.

- gui:

  A character string. This formal argument exists for compatibility
  reasons only.

- browser:

  A character string. This formal argument exists for compatibility
  reasons only.

- remote:

  A character string. This formal argument exists for compatibility
  reasons only.
