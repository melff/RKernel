# Remove a Class to the 'Paged' Ones

Remove a class from those who are output using
[`Page()`](https://melff.github.io/RKernel/reference/Page.md) when they
are autoprinted, i.e. returned as the value of the last expression in a
Jupyter notebook cell.

## Usage

``` r
remove_paged_classes(x)
```

## Arguments

- x:

  A character string, the name of a class.
